A fully functioning E-commerce web application developed using:
> PHP
> CSS (bootstrap)
> Javascript
> HTML
> MySQL

Functionality:
> Product browsing (with pagination)
> User/customer Registration (with authentication checks)
> User/customer/admin Login
> User/customer/admin Logout
> User session creation
> Customer purchasing products
> User uploading products for purchase
> Customer/admin checkout
> Admin control panel
> Admin product control panel
